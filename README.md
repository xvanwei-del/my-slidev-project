# 迭代算法 —— 算法设计与分析 (Slidev Presentation)

这是一个基于 [Slidev](https://sli.dev/) 开发的演示文稿项目，主题为**迭代算法**。该文稿结合了学术内容与现代化的交互式 UI 设计，旨在深入浅出地讲解算法设计中的核心迭代思想。

## 📖 内容概览

本项目详细探讨了以下算法及问题：
- **斐波那契数列 (Fibonacci Sequence)**：迭代与递归的对比分析。
- **最大公约数 (GCD)**：欧几里得算法的迭代实现。
- **牛顿迭代法 (Newton's Method)**：数值分析中的经典根寻找算法。
- **穿越沙漠问题 (Desert Crossing Problem)**：动态规划与迭代思想的结合应用。

## ✨ 特性

- 📝 **Markdown 驱动**：使用 `slides.md` 编写内容，简单高效。
- 🎨 **高度可定制**：通过 `custom.css` 实现了独特的视觉风格（融入了流体感与科技感元素）。
- 🚀 **即时预览**：支持热更新，修改内容后页面自动刷新。
- 📊 **交互组件**：集成代码高亮、数学公式及动态图表。

## 🛠️ 本地开发

确保你已经安装了 [Node.js](https://nodejs.org/)。

1. 克隆项目：
   ```bash
   git clone https://github.com/xvanwei-del/my-slidev-project.git
   cd my-slidev-project
   
2. 安装依赖：
Bash
npm install

3. 启动开发服务器：
Bash
npm run dev
启动后，在浏览器访问 http://localhost:3030 即可查看幻灯片。

📦 构建与部署

生成静态网页
如果你想生成可以直接部署在网页上的 HTML 文件：

Bash
npm run build
构建出的文件将存放在 dist 目录中。

导出为 PDF
Bash
npm run export
